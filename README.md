# QueuePro
QueuePro is a full-stack, web-based Smart Queue Management System designed specifically for Government Service Offices. It digitizes and automates the entire token-based queue handling process across government departments including Passport Services, Driving License (RTO), Tax Services, and Certificate Issuance offices.

The system eliminates the need for citizens to physically stand in long queues without any visibility into their waiting time. Using a real-time, role-based architecture, QueuePro ensures that citizens, officers, and administrators each have a tailored experience with appropriate access and controls.

Built on Node.js and Express.js with a MongoDB database, the application follows strict MVC (Model-View-Controller) architecture principles, separating business logic into Controllers, data definitions into Models, API routing into Route files, and UI rendering into EJS Views. Authentication is handled via JSON Web Tokens (JWT) stored in HttpOnly cookies for security.
